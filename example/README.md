# Resource Mapper Example

This is an example project demonstrating how to use `@resource-mapper/core` to generate service architecture documentation.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start development server:
   ```bash
   npm run dev
   ```

3. Generate static site:
   ```bash
   npm run generate
   ```

The generated site will be in `.resource-mapper/out` and can be deployed to any static hosting service.

## Project Structure

- `services/` - Service definitions organized by groups
- `teams/` - Team definitions
- `resource-mapper.config.json` - Configuration file
